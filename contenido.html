Editores de Texto

EDITORES DE TEXTO


Sublime Text

Para algunos, Sublime Text marca el listón cuando se trata de editores de texto. Es un hermoso editor de texto con muchas funciones para editar código que destaca la experiencia del usuario. Sus características incluyen un modo de escritura sin distracciones y edición dividida, además de accesos directos rápidos y búsqueda.

Atom

Atom es multiplataforma y se basa en el marco electrónico, por lo que los codificadores negativos que prefieren que sus aplicaciones sean livianas pueden oponerse a su tamaño de instalación relativamente grande.

Visual Studio Code

Visual Studio Code tiene su propio terminal y depurador, admite linting y tiene integración con todo tipo de herramientas de control de código fuente. Creemos que es uno de los mejores IDE para desarrolladores de Python, ya que sugiere finalizaciones y proporciona ventanas emergentes sobre la marcha que muestran la documentación de clases y métodos.

Notepad++

Al igual que otras soluciones de nuestra lista, Notepad ++ es un proyecto de código abierto en el que cualquiera puede contribuir.

Vim

Con soporte para Windows, Linux y Mac, el editor de texto de Vim es un editor confiable y estable, además de que este puede integrarse a otras populares herramientas. Este fue diseñado para ser usado desde una interfaz de línea de comando (CLI) y como una aplicación única en una interfaz de usuario gráfica (GUI).

VERSIONES DE HTML


HTML 2.0

En 1995 se publica el estándar HTML 2.0. A pesar de su nombre, HTML 2.0 es el primer estándar oficial de HTML, es decir, el HTML 1.0 no existió como estándar. HTML 2.0 no soportaba tablas. Se simplificaba al máximo la estructura del documento para agilizar su edición, donde la declaración explícita de los elementos body, html y head es opcional.

HTML 3.2

La versión HTML 3.2 se publicó en 1997 y es la primera recomendación de HTML publicada por el W3C (Consorcio internacional). Esta revisión incorporó los últimos avances de las páginas web desarrolladas hasta 1996, como applets de Java y texto que fluye alrededor de las imágenes.

HTML 4.01

La última especificación oficial de HTML se publicó en diciembre de 1999 y se denomina HTML 4.01. Desde la publicación de HTML 4.01, el W3C se centró en el desarrollo del estándar XHTML.

HTML 5, HTML 5.1, HTML 5.2

Hay nuevas etiquetas, se introduce la posibilidad de introducir audio y video de forma directa en la web sin necesidad de plugins o complementos en los navegadores, y otras novedades. El W3C irá lanzando progresivamente nuevas evoluciones del HTML 5.
