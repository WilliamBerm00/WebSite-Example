Introducción

Al diseñar un sitio web, HTML es uno de los aspectos más importantes a tener en cuenta. Los comandos de codificación HTML son mucho más simples que los de JAVA, PHP, .net, etc. Además, varios navegadores de Internet están diseñados para leer HTML fácilmente. Diseñar un sitio web no es una tarea muy difícil si una persona sigue un procedimiento sistemático. Si no se siguen los procedimientos, puede resultar mucho más difícil que la ciencia espacial.

Los elementos HTML están representados por etiquetas. Las etiquetas HTML identifican elementos de contenido como "encabezado", "título", "párrafo", "tabla", etc.

- William Bermudez Quiroga
